1.Which function extracts a substring?

a) TO_CHAR()

b) SUBSTR()

c) NVL()

d) ROUND()


Answer:b) SUBSTR()



2.Which function is used to count the number of rows in a table?

a) COUNT(*)

b) SUM(*)

c) ROW_COUNT()

d) TOTAL()


Answer: a) COUNT(*)


3.What will NVL(salary, 0) do in SQL?

a) Nullify salary values

b) Replace all null salary values with 0

c) Replace salary with NULL

d) Replace 0 with NULL


Answer: Replace all null salary values with 0


4.Which group function returns the largest value in a column?

a) MIN()

b) SUM()

c) MAX()

d) COUNT()


Answer: c) MAX()


5.The SUBSTR function in SQL is used to:

a) Replace a substring with another string

b) Extract a part of a string

c) Convert numbers to strings

d) Remove spaces from a string

Answer:  Extract a part of a string

21.Which of the following SQL statements will retrieve the highest salary
